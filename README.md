# 🔢 NumPy Data Explorer

A hands-on exploration of NumPy — the core numerical computing library in Python — covering array operations, statistical analysis, reshaping, broadcasting, and performance benchmarking.

> 📌 **Syntecxhub Data Science Internship | Week 1 — Project 1**

---

## 📖 About the Project

This project demonstrates the fundamentals of NumPy through a structured, well-documented Jupyter Notebook. It covers everything from basic array creation to performance comparisons with standard Python lists — building the foundation needed for data science and machine learning workflows.

---

## ✅ What's Covered

| Section | Topics |
|---|---|
| Array Creation | `np.array`, `zeros`, `ones`, `eye`, `arange`, `linspace`, `random` |
| Indexing & Slicing | Row/column access, boolean masking, step slicing |
| Math Operations | Element-wise, axis-wise (`axis=0/1`), aggregate |
| Statistical Analysis | `mean`, `median`, `std`, `min`, `max`, `percentile` |
| Reshaping & Broadcasting | `reshape()`, `flatten()`, broadcasting rules |
| Save & Load | `.npy`, `.npz`, `.csv` formats |
| Performance Benchmark | NumPy vs Python lists — speed & memory comparison |

## 🚀 How to Run

1. Clone the repository
```bash
git clone https://github.com/yourusername/Syntecxhub_NumPy_Data_Explorer.git
cd Syntecxhub_NumPy_Data_Explorer
```

2. Install dependencies
```bash
pip install numpy jupyter
```

3. Launch the notebook
```bash
jupyter notebook project1_numpy_data_explorer.ipynb
```

4. Run all cells with **Kernel → Restart & Run All**

## 💡 Key Takeaways

- NumPy arrays are significantly faster and more memory-efficient than Python lists
- Broadcasting eliminates the need for explicit loops in most numerical operations
- Axis-based operations allow flexible row-wise and column-wise computations
